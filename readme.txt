Set the product link in the WooCommerce cart page to a page/post, than redirect to the default single product page.


== Description ==

By default, product default HTML link in the WooCommerce Cart Page will be linked to its single product page.  This might not be desirable when you have actually created an individual page/post for the product and wish to redirect to this page instead.

This plugin is to allow you to change the product default HTML link in the WooCommerce Cart Page to another page/post. 

You are free to set it to any published page or post. By default, it's will still redirect to single product page.


== Installation ==

This section describes how to install the plugin and get it working:

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress



== Frequently Asked Questions ==

This plugin require no settings and configuration, once activated, it will show up in the WooCommerce Product > General Tab


== Screenshots ==

1. Default product link in Cart Page is single product page. (e.g. your-domain.com/?product=pumpkin)
2. Set the Cart Link to a new page, for example, we set it to "News". Save and update the Product.)
3. Refresh the Cart Page to see now it will link to "news" page! (e.g. your-domain.com/news).
